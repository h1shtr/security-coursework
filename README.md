# Security Coursework

A collection of labs and design projects from my Forensic Computing & Cyber Security degree
at the University of Prince Mugrin.

| Folder | Course | What it is |
|---|---|---|
| [`honeypot-os-security/`](honeypot-os-security) | FC254 — OS Security | A deliberately weakened Linux VM instrumented with OSSEC, auditd, Suricata, ClamAV, and Falco, plus custom SSH alert/recording scripts. |
| [`privilege-escalation-lab/`](privilege-escalation-lab) | FC382 — Defense Mechanisms | A full attack chain (PHP command injection → DirtyPipe / SUID escalation → root) detected end-to-end with OSSEC. |
| [`alumni-networking-app-security-design/`](alumni-networking-app-security-design) | FC331-332 — Software Security | A secure-by-design case study: architecture, STRIDE threat modeling, and DREAD risk scoring for a mentorship platform. |

Each folder has its own README with full details.
